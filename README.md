# AI/ML Internship Tasks

## Overview

This repository contains the tasks completed during the AI/ML Internship Program. The projects cover Exploratory Data Analysis (EDA), Regression, Classification, Prompt Engineering, Large Language Models (LLMs), and Machine Learning applications in healthcare, finance, and real estate.

---

## Task 1: Iris Dataset Exploratory Data Analysis (EDA)

### Objective

Perform Exploratory Data Analysis on the Iris dataset to understand feature distributions and relationships among different Iris flower species.

### Dataset Used

* Iris Dataset

### Models Applied

* Exploratory Data Analysis (EDA)
* Data Visualization using Matplotlib and Seaborn

### Key Results & Findings

* Setosa species formed a clearly distinct cluster.
* Petal length showed strong separation among species.
* Box plots revealed variation in petal width across species.
* Visualizations helped identify patterns and feature relationships.

## Task 2: Stock Price Prediction

### Objective

Predict short-term stock closing prices using historical stock market data.

### Dataset Used

* Apple Inc. (AAPL) historical stock data retrieved using Yahoo Finance (yfinance API)

### Models Applied

* Linear Regression

### Key Results & Findings

* Used Open, High, Low, and Volume as predictive features.
* Achieved strong prediction performance for short-term forecasting.
* Actual and predicted stock prices followed similar trends.
* Demonstrated the effectiveness of Linear Regression for short-term stock prediction.

## Task 3: Heart Disease Prediction

### Objective

Develop a machine learning model to predict the risk of heart disease using patient health data.

### Dataset Used

* Heart Disease UCI Dataset

### Models Applied

* Logistic Regression

### Key Results & Findings

* Successfully classified patients into heart disease and non-heart disease categories.
* Age and chest pain type were identified as important predictive factors.
* Achieved strong Precision, Recall, and Classification Accuracy.
* ROC-AUC analysis demonstrated reliable model performance.
